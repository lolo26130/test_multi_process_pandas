## Project Overview

A **PyQt6 real-time audio acquisition and visualization application** that demonstrates multiprocessing with shared memory, zero-copy pandas DataFrames, and thread-safe Qt signaling. The project uses `uv` as its package manager (lock file present).

## Commands

```bash
# Install (uv preferred)
uv pip install -e .

# Run the application
python -m test_multi_process_pandas.main

# Run all tests (headless, offscreen rendering configured automatically)
pytest tests/

# Run a single test file
pytest tests/test_acq_worker.py -v

# Build Sphinx documentation
cd docs && make html
```

Tests require `pytest-qt` and run headless via `QT_QPA_PLATFORM=offscreen` (set in `pyproject.toml`).

## Architecture

The application connects three concurrent execution contexts:

```
acq_worker (subprocess)
  ‚Üí writes audio frames to SharedMemory circular buffer
  ‚Üí puts write-index into mp.Queue

DataWatcher (QThread)
  ‚Üí polls mp.Queue with 100ms timeout
  ‚Üí emits data_ready(int) signal (queued connection)

MainWindow (Qt GUI thread)
  ‚Üí receives data_ready, reads buffer via get_views()
  ‚Üí updates pyqtgraph plots (throttled to 30 Hz)
  ‚Üí updates QTextEdit log (throttled to 2 Hz)
```

**Circular buffer** (`BUFFER_SIZE = 96000` frames = 2 s at 48 kHz, 4 columns: `t, ch1, ch2, ch3`). The `get_views()` method returns 1 or 2 numpy/pandas views (chronological order, handling wrap-around) with zero-copy ‚Äî the DataFrame wraps the SharedMemory array directly.

**Key constants** (all in `main.py:31‚Äì59`): `SAMPLE_RATE=48000`, `CHUNK_SIZE=1024`, `BUFFER_SIZE=96000`, `PLOT_PERIOD=1/30`, `LOG_PERIOD=0.5`.

## Key Design Points

- **`acq_worker` accepts `_stream_factory`** for dependency injection in tests ‚Äî `FakeInputStream` in `conftest.py` replaces `sounddevice.InputStream` without touching production code.
- **UI file workflow**: edit `views/main.ui` ‚Üí run `pyuic6 main.ui -o ui_main.py` ‚Üí `test_compile_ui.py` verifies it stays in sync.
- **Graceful shutdown order**: set `_closed` flag ‚Üí stop DataWatcher thread ‚Üí terminate subprocess ‚Üí unlink SharedMemory. Skipping steps causes `ResourceWarning` or zombie processes.
- **`setup.cfg`** still has stale metadata (name `AbsorptionLaser`, version `1.0`). `pyproject.toml` is the authoritative build config.
